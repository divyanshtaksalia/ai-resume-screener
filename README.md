# 🚀 AI Resume Screener (FastAPI + Gemini 1.5/2.5 Flash)

A modern, AI-powered application that analyzes resumes against job descriptions to provide a match score, identifies key skills, and highlights missing requirements.

## ✨ Features
- **AI Analysis:** Uses Google Gemini (2.5 Flash) for semantic matching.
- **PDF Parsing:** Automatically extracts text from uploaded PDF resumes.
- **Modern UI:** Responsive dashboard built with Tailwind CSS and Frontend Codex.
- **Real-time Feedback:** Provides a Match Score, Matched Keywords, and a Professional Summary.

## 🛠️ Tech Stack
- **Backend:** FastAPI (Python)
- **AI Model:** Google Gemini API
- **Frontend:** HTML5, Tailwind CSS, JavaScript (Fetch API)
- **Libraries:** `PyPDF2`, `google-generativeai`, `python-multipart`
